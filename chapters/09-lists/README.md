# Chapter 9: Lists

Boot.dev "Learn to Code in Python" - Chapter 9

## Chapter Goal
Understand Python list operations for storing, transforming, and filtering collections of data.

## Lesson Concepts Covered
- List basics and list continuation concepts
- Counting and indexing into lists
- List length and updates
- Appending and popping values
- Counting items in a list
- No-index iteration style
- Finding items and max/increase patterns
- Modulo usage in list workflows
- Slicing lists
- Concatenate and contains operations
- List deletion
- Tuples and first-element access
- Reverse and filter-style practice

## Security Mapping
- **OWASP A09: Security Logging and Monitoring Failures**
- List operations support event aggregation, filtering, triage, and report-ready outputs.

## Portfolio Application
- Build list-based event queues and suspicious-IP candidate sets.
- Use slicing/filtering to produce concise incident summaries.
