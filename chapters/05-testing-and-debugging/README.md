# Chapter 5: Testing and Debugging

Boot.dev "Learn to Code in Python" - Chapter 5

## Chapter Goal
Develop a repeatable test/debug workflow to catch logic bugs and fail safely.

## Lesson Concepts Covered
- Unit tests and why they matter
- Two lesson/practice styles
- Debugging workflow basics
- Learning loops and correction cycles
- Debugging practice drills
- Stack trace interpretation

## Security Mapping
- **OWASP A05: Security Misconfiguration**
- Tested code and controlled debugging reduce unsafe failure behavior and information leakage.

## Portfolio Application
- Add test-first checks for parser and validator functions.
- Ensure user-facing errors are concise while preserving internal diagnostics.
